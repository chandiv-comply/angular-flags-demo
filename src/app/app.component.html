Angular Countries Flags

A super simple angular library for displaying country flag.

Based on ISO 3166-1 alpha-2 list of countries.

  • Version: Latest
  • Author: Chandi Charan
  • Created: 28 December, 2021
  • Updated: 28 December, 2021

If you have any questions that are beyond the scope of this documentation Please feel free to email at chandicharan5983@gmail.com.

Installation & Usage

Follow the steps below to install the package in your angular project

  1. Run npm i angular-countries-flags in your angular project.
  2. Import import {AngularCountriesFlagsModule} from angular-countries-flags in the module you want to use the flag component.
  3. Put AngularCountriesFlagsModule in imports array of the module. 
                                      
                                  @NgModule({
                                    declarations: [
                                      AppComponent
                                    ],
                                    imports: [
                                      BrowserModule,
                                     AngularCountriesFlagsModule 
                                    ]
                                    bootstrap: [AppComponent]
                                    })
  4. Put the markup of the component in the component template
    <flag [code]="'IN'" [size]="'lg'" [effect]="'app'"></flag>

    The Above code will generate Indian Flag

Available Configurations

@Input

Property & type Valid Values Is Required
code : string ISO 3166-1 alpha-2 country code Yes
size : string 'sm' , 'md' , 'lg' , 'xl' No (default to 'sm')
effect : string 'round','sphere','wave','app' No

If you need more information, please visit bootstrap site: https://getbootstrap.com
